# Technical Exercise - Bowling Game

This exercise creates a simple bowling game and lets users play through the console


## Installation

* Install dependencies:

  ```
  $ bundle install
  ```

## Usage

Start the game by running:

  ```
  $ bin/bowling
  ```

## Testing

Unit tests are implemented with RSpec:

  ```
  $ bundle exec rspec
  ```
